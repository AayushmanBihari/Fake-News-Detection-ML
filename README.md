📰 Fake News Detection using Machine Learning

Overview

This project detects whether a news article is Real or Fake using Machine Learning and Natural Language Processing (NLP).

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- TF-IDF Vectorizer
- Multinomial Naive Bayes
- Google Colab
- Jupyter Notebook

Features

- Data Preprocessing
- Text Cleaning
- TF-IDF Feature Extraction
- Machine Learning Model Training
- Real/Fake News Prediction

Dataset

- Synthetic Real News Dataset (5,000 records)
- Synthetic Fake News Dataset (5,000 records)
